Assembly Overview

This document outlines the major steps in assembling the Mac Classic Fruit Jam build: cable prep, securing breakout boards, and display mounting.

Parts

  • 5" HDMI Display
  • Fruit Jam PCB
  • USB-C Breakout
  • Right-angle HDMI adapters ×2
  • Jump wires
  • Machine screws (M2.5 / M3)
  • Rubber bumpers

Prepare HDMI Cable

  1. Open the latch on each HDMI adapter.
  2. Insert cable with correct orientation.
  3. Close latch securely.

USB-C Breakout

  1. Tin GND and VBUS pads.
  2. Solder jumper wires.
  3. Mount breakout using M2.5 screws.

Secure Display

  1. Align display to printed frame.
  2. Fasten using M3 × 6mm screws.
  3. Ensure cable clearance.

SD Adapter Disassembly

  1. Carefully remove plastic enclosure.
  2. Expose PCB for fitment.

Finishing Steps

  1. Add rubber bumpers.
  2. Verify system boots + display works.
