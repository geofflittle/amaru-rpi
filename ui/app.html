Assembly

Short intro: this section walks through gathering parts and assembling the Mac Classic Fruit Jam. The steps are organized into logical blocks: prep cables, secure breakouts, mount PCB, attach the display, and finishing touches.

Parts

Gather these parts and the listed fasteners before starting the build.

  • 5" HDMI display
  • Fruit Jam PCB
  • USB-C breakout
  • Right-angle HDMI plug adapters ×2
  • Assorted machine screws (M2.5, M3)
  • Rubber bumpers

Prep HDMI cable

  1. Open the release latch on each right-angle HDMI adapter.
  2. Insert the HDMI ribbon cable into each adapter using the flex clips.
  3. Verify the cable is firmly seated and the latch is closed.

USB-C Breakout & Wiring

  1. Select two silicone jumper wires (example: red and blue).
  2. Trim one end of each and tin the wires for soldering.
  3. Solder the ground wire to GND and the power wire to VBUS on the breakout.
  4. Secure the breakout to the printed USB mount with M2.5 × 6mm screws.

Secure 5" Display

Mount the display to the 3D printed display mount and secure using M3 × 6mm screws.

  1. Orient the display so ports line up with cutouts.
  2. Fasten the display with four M3 × 6mm machine screws.
  3. Confirm the display sits flush and all cables reach their connectors.

SD Adapter Prep

Disassemble the SD card adapter's plastic enclosure to extract the PCB if required for this build.

  1. Use a prying tool to carefully open the enclosure.
  2. Remove the PCB and set the plastic shells aside (they won't be reused).

Finishing

Final checks and installing rubber feet.

  1. Attach rubber bumpers to the bottom cover using adhesive.
  2. Perform a final test: power on and verify HDMI, USB power, and SD card behavior.
